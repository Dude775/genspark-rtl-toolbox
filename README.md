# Genspark Download Toolbox

תוסף לניהול והורדת שיחות מ-Genspark.ai עם פיצ'ר חיפוש מתקדם.

## מה חדש בגרסה האחרונה

- הסרת אלגוריתם RTL במלואו.
- תיקון כפילויות בהורדות (TXT, JSON, JSON+TXT).
- הוספת חיפוש טקסט מתקדם עם הדגשה וגלילה להודעות.
- ממשק משתמש משודרג בפופאפ.
- שם התוסף שונה ל-"Genspark Download Toolbox".

## התקנה

1. הורד או שכפל את התוסף.
2. טען כתוסף בדפדפן דרך מצב מפתחים.
3. השתמש בממשק להורדות ולחיפוש.

## תצורה

ניתן להגדיר הגדרות ברירת מחדל ב-background.js.

## תרומות

ברוכים הבאים לתרום ולדווח באגים.
